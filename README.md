# ieru
心理的安全性を高め発言のハードルを下げるチャットツール.

url : [https://ieru.life](https://ieru.life)

# Setup

```bash
$ git clone https://github.com/addtobasic/ieru.git
$ cd ieru
$ yarn install
```


上のコマンドを実行した後にconfig/firebase.ts に習って.env.localを作成し, firebaseの環境変数を設定をします.

# Start

```bash
$ yarn dev
```
